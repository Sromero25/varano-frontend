# Varano - Frontend Admin

Panel de administración para gestión de calzado deportivo.

## Tecnologías
- Angular 17
- Bootstrap 5
- SCSS
- TypeScript

## Instalación
1. `npm install`
2. `ng serve`
3. Abrir http://localhost:4200

## Características
- Gestión de productos, categorías y marcas
- Control de inventario
- Gestión de pedidos y clientes
- Dashboard con estadísticas
